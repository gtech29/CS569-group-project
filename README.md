
# AI-Powered Game Design Insight Tool

## Overview

This project examines how game developers can be assisted using artificial intelligence by analyzing the available games and coming up with usual design patterns.

Rather than having to go through dozens of games, seeing the forum chats, or even viewing the gameplay footage manually, the tool gathers information that is publicly available and helps to organize it into short, formatted information.

The main goal is to enable developers to take wise design decisions at an earlier stage of development and yet retain complete creative power.

This repository is our demonstration of a prototype developed in the course of our COMP 569 studies.

---

## The Problem

Independent and student game developers usually have to make use of dispersed information including:

- Blog posts
- Game reviews
- Developer interviews
- Community discussions

It can be extremely time-consuming to find coherent instructions on use of systems such as inventory management, tutorials, dialogue systems, or quality of life features.

Significant usability choices are occasionally found deep into the development cycle when they become more painful and costly to address.

No centralized tool that analyses existing games in a systematic way to come up with structured design guidance is currently available.
